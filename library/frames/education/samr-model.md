# SAMR Model

## Overview

The SAMR Model sorts technology-enabled instructional changes into four levels based on the degree of pedagogical transformation—from simple tool replacement to fundamental redesign of what is possible. Its core claim is that **not all technology integration is equally valuable**: substituting a tool does not improve learning, while modification and redefinition unlock instructional possibilities that were not available before. The model helps educators distinguish between digitizing existing practice and actually transforming it, which is essential for evaluating whether a technology investment is worth the effort.

## Categories

1. **Substitution**
   - Technology is used as a **direct substitute** for a traditional tool or medium, with no functional change to the task itself.
   - The instructional objective, assessment, and pedagogical pattern remain identical; only the delivery medium changes.
   - Discriminating criterion: if you removed the technology, the lesson would proceed in exactly the same way, just slower or on paper.
   - Example: typing an essay instead of handwriting it; using Google Slides instead of poster board for a presentation.

2. **Augmentation**
   - Technology **adds functional improvement** to the traditional task without changing the fundamental nature of the activity.
   - The same learning outcome is still the goal, but the tool provides speed, capacity, or convenience that enhances execution.
   - Discriminating criterion: the task is noticeably better or faster with the technology, but a skilled practitioner could accomplish the same end without it (just with more friction).
   - Example: using a calculator to check arithmetic (vs. doing it by hand); using spell-check while writing.

3. **Modification**
   - The task is **significantly redesigned** in scope or complexity because of what the technology affords; the learning objective shifts measurably.
   - The technology enables a fundamentally different approach that was not practical before, and students engage with the content in a new way.
   - Discriminating criterion: the presence of technology **changes what students are asked to do**, not just how they do it; the assessment itself would be different.
   - Example: students analyzing real-time climate data to construct their own arguments about global trends (vs. reading a textbook about climate science); using video annotation tools to analyze media in ways that require frame-by-frame reasoning.

4. **Redefinition**
   - Technology allows for tasks that were **previously inconceivable** without it; it opens entirely new learning possibilities.
   - The technology is not an enhancer of an existing activity—it creates a fundamentally new task or form of engagement that was not possible in a pre-digital context.
   - Discriminating criterion: there is no meaningful offline analog; the learning activity is native to the technology and could not exist in traditional form.
   - Example: students collaborating synchronously with peers in another country in a shared virtual environment; publishing student work to a real global audience and receiving feedback from people outside the classroom.

## Classification Procedure

1. **Describe the instructional task.** Write down what students are asked to do, the learning objective, and how they would accomplish it without the technology in question.
2. **Ask: Would the task be identical without the technology?**
   - If **yes** → Substitution.
   - If **no**, proceed to step 3.
3. **Ask: Is the learning objective the same, but the task is faster, easier, or more accessible?**
   - If **yes** → Augmentation.
   - If **no**, proceed to step 4.
4. **Ask: Has the scope or complexity of the task changed such that students are now doing something fundamentally different than they would without the technology?**
   - If **yes** → Modification.
   - If **no**, return to step 3 (likely Augmentation).
5. **Ask: Is this task possible at all without the technology? Could an offline version exist?**
   - If **no** — it is impossible offline → Redefinition.
   - If **yes but awkward** — it is theoretically possible but impractical → likely Modification.
6. Record the classification and note whether the technology use is worth the implementation cost: Redefinition and Modification typically justify professional development and infrastructure investment, while Substitution usually does not.

## Implications per Category

| Category | Pedagogical status | Implementation guidance |
|---|---|---|
| **Substitution** | No transformation; digitization only. | Low priority. Tech skills may matter, but learning outcomes are unchanged. Do not invest heavily. |
| **Augmentation** | Efficiency gain, not rethinking. | Medium priority if the efficiency gain is significant (time freed for higher-order work); low priority if marginal. |
| **Modification** | Measurable redesign of the task and learning. | High priority. Invest in training and time; the change in what students do justifies the effort. |
| **Redefinition** | Unlocks new learning possibilities. | Highest priority. This is where technology integration pays its highest dividends; plan for sustained professional development and iteration. |

Implicit strategic implication: **Prioritize Modification and Redefinition uses; audit existing Substitution initiatives for opportunity cost; use Augmentation to fill gaps where efficiency matters but transformation is not the goal.**

## Common Misclassifications

- **Substitution mistaken for Augmentation.** Assuming that using technology automatically improves learning because it feels more modern. The tell is that the learning outcome, task structure, and assessment remain identical. Consequence: wasted training time and infrastructure cost with no measurable gain.
- **Augmentation mistaken for Modification.** Treating a speed or convenience improvement as if it has changed the fundamental learning activity. The tell is that the lesson objective, what students produce, and how they would demonstrate mastery are the same. Consequence: over-investing in a change that only saves time.
- **Modification mistaken for Redefinition.** Assuming that any task that requires new technology is inherently new. The tell is that a reasonable offline version of the task could exist, just with more friction. Consequence: misallocating scarce professional development resources toward changes that could be accomplished without technology.
- **Redefinition mistaken for Modification.** Underestimating how truly new a task is because it seems like a small change in the user interface. The tell is that the learning task literally did not exist before the technology became available. Consequence: under-investing in training and iteration time, then blaming the technology when adoption stalls.
- **Classifying technology in isolation from the task.** A tool (e.g., a document editor) is neutral; the classification depends on what the teacher asks students to do with it. The same tool can enable Substitution in one lesson and Modification in another.
