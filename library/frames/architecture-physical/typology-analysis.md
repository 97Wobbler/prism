# Typology Analysis (Rossi)

## Overview

Typology Analysis sorts urban buildings and districts by their **morphological persistence** and **role in collective urban memory**. Rossi's core claim is that cities are not random accumulations but structured by recurring types — dwelling, market, civic building, street pattern — that persist across centuries and geography. The practical implication is that different types have different design and preservation logics: a primary type that shapes urban structure demands different intervention than a secondary type or a contingent infill. Understanding typology constrains which analytical tools and design responses are appropriate.

## Categories

1. **Primary Type** (Type-as-structure)
   - A building or spatial organization that **fundamentally shapes urban form** and has persisted across multiple historical periods.
   - The type is recognized as structuring because removing it or fundamentally altering it destabilizes the surrounding fabric.
   - Discriminating criterion: the building or pattern establishes a rule that other elements conform to; it is generative of form, not conforming to it.
   - Example: the Roman grid in cities like Turin or Barcelona; the plaza mayor pattern in Spanish colonial towns; the street arcade in Bologna.

2. **Secondary Type** (Type-as-practice)
   - A building type that is **widely repeated and recognized** within a culture or period but is not load-bearing to the urban structure — it fills a repeated social or functional need.
   - The type is durable enough to appear in multiple instances, but the city's form would reorganize rather than collapse if the type disappeared.
   - Discriminating criterion: the type serves a consistent function (residential block, market hall, school) and has recognizable morphological features, but is subordinate to primary types in generating structure.
   - Example: the 19th-century residential block in Paris; the Victorian townhouse row; the modernist apartment slab of the 1950s–70s.

3. **Tertiary Type** (Type-as-convention)
   - A building or object that follows recognizable conventions and is **culturally legible** but arises from habit or regulation rather than from a deep structural principle.
   - These types are numerous, replaceable, and their loss does not alter urban form.
   - Discriminating criterion: the type is readable as a category (e.g., corner shop, bus shelter) but is contingent to the place; the same urban structure would accommodate a different tertiary type.
   - Example: shopfronts, kiosks, telephone booths, traffic barriers, temporary markets.

4. **Singular Object** (Type-negative)
   - A building or intervention that **breaks from or does not conform to any recognizable type** — it is intentionally or accidentally singular, monumental, or anomalous within its context.
   - It may be memorable, but its memorability comes from difference, not from participation in a persistent category.
   - Discriminating criterion: the object cannot be classified as an instance of a recurring type; to describe it requires specific reference to its own program or author.
   - Example: an iconic museum by a named architect; a unique civic monument; a building whose formal gesture is deliberately non-repeatable.

5. **Obliterated or Latent Type**
   - A type that **existed and shaped urban form in the past** but has been systematically destroyed, replaced, or is no longer recognizable in the current fabric.
   - The type may still be legible through traces, excavation, or historical record, but it no longer actively structures contemporary experience.
   - Discriminating criterion: the type is recoverable through historical analysis but is absent or nearly absent from the physical present.
   - Example: the medieval street pattern underlying a 19th-century replan (visible in property lines); demolished courtyard housing replaced by tower blocks; lost industrial typologies.

## Classification Procedure

1. Identify the building, district, or spatial pattern in question. Document its **morphological features** (form, material, height, street relationship, plan type) and **functional program** (what it does, who uses it, how consistently).

2. Ask: **"Does this element shape how other elements are organized around it, or does it conform to a larger structure?"**
   - If it shapes the structure (e.g., a plaza that streets converge on, a street grid that blocks conform to) → likely Primary Type. Go to step 4.
   - If it conforms to a larger structure → go to step 3.

3. Ask: **"Is this recognizable as an instance of a type that appears multiple times in this city or culture, and is it functionally persistent (serving a durable social need)?"**
   - If **yes, and it is load-bearing to district form** → Secondary Type. Go to step 4.
   - If **yes, but it is infill or contingent** (could be replaced without restructuring) → Tertiary Type. Go to step 4.
   - If **no** (singular, anomalous, or unclassifiable) → Singular Object. Go to step 4.

4. Ask: **"Is this type present and legible in the current physical fabric, or is it absent/obliterated?"**
   - If present → the classification stands (Primary, Secondary, Tertiary, or Singular).
   - If absent but recoverable through historical evidence → Obliterated or Latent Type.

5. Record the classification, note which types neighboring elements belong to, and state the **design or preservation implication** (see Implications below).

## Implications per Category

| Category | Design/Preservation Logic |
|---|---|
| **Primary Type** | Respect and reinforce the type-rule; new interventions should strengthen its legibility, not subvert it. Alteration or removal is a major urban surgery requiring justification at the city-structure scale. |
| **Secondary Type** | Evolution and substitution are acceptable; these types serve a function, and if program or need shifts, the type can be replaced by a new secondary type. Maintain morphological coherence with primary types. |
| **Tertiary Type** | Low barrier to change; optimize for current use and convenience. These types should support primary and secondary types, not compete with them for attention. |
| **Singular Object** | Document and protect; singular objects are non-recoverable and cannot be replaced by substituting a type. Their intervention requires author/architect-level decision-making, not type-based rules. |
| **Obliterated or Latent Type** | Recover or reinterpret if the type is foundational to urban identity; treat obliteration as a historical loss. Reintroduction of a latent type can stabilize district form if done with historical literacy, not fantasy. |

## Common Misclassifications

- **Secondary mistaken for Primary.** Treating a widespread residential type (e.g., 19th-century apartment blocks) as if it were generative of urban structure when it is actually infill. The tell is that the city's skeleton persists if the type is removed; only density or character changes. Consequence: over-protecting generic housing stock and under-investing in actual structure.

- **Primary mistaken for Singular.** Assuming a famous or architecturally distinguished building is singular when it is actually an instance of a primary type (e.g., a cathedral as an instance of the civic-building type). The tell is that removing or radically altering it destabilizes the district form. Consequence: missing the type-rule and designing a replacement that breaks the structure.

- **Tertiary mistaken for Secondary.** Elevating a contingent infill (e.g., a row of shopfronts) to the status of a type that "must be preserved." The tell is that the object serves a passing need and can be functionally replaced without loss of urban coherence. Consequence: freezing the city and preventing adaptation.

- **Obliterated type treated as fictional.** Recovering a latent type from historical record and reintroducing it without acknowledging that the urban context has changed, leading to pastiche. The tell is that the reintroduced type clashes with current primary and secondary types. Consequence: inauthentic heritage and wasted investment.

- **Singular Object classified as Secondary by formal resemblance.** A unique building that happens to look like a recurring type (e.g., a modernist tower that resembles apartment slabs) but serves a unique program or carries singular historical weight. The tell is that the building's significance lies in its specific authorship or cultural moment, not in its replicability. Consequence: generic treatment of a non-generic building.
