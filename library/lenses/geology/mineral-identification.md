# Mineral Identification

## Overview

Mineral identification combines five observable properties—hardness, cleavage, luster, streak, and crystal form—applied in a specific order to narrow an unknown specimen to a likely candidate. The discipline is *observation precision*: most misidentifications result from either skipping properties (relying on color alone) or conflating visual impression with measurement. Practitioners use this when they encounter an unknown sample in the field or lab and need to determine what it is before investing in costly analysis.

## Analytical Procedure

### Step 1 — Establish the specimen's color and general appearance

1. **Examine the sample in daylight** (or equivalent neutral lighting). Record the color as seen in daylight.
2. **Note any color variation.** Is it uniform, banded, mottled, or iridescent? Does the color appear to be in the mineral itself or on the surface (tarnish, weathering)?
3. **Assess transparency.** Classify as opaque, translucent, or transparent. Hold it to light if it is small enough.
4. **Record any visible crystal form.** Note if the specimen shows faces, edges, or growth patterns. Do not assume crystal form is diagnostic yet—many minerals have the same form.

### Step 2 — Measure hardness (Mohs scale)

Hardness is the first *quantitative* test and eliminates large swaths of possibilities.

1. **Obtain a hardness reference kit** or use common materials:
   - Mohs 1: Talc (mark with fingernail)
   - Mohs 2: Gypsum (fingernail scratches it easily)
   - Mohs 3: Calcite (copper coin scratches it)
   - Mohs 4: Fluorite (steel knife scratches it easily)
   - Mohs 5: Apatite (steel knife scratches it with moderate pressure)
   - Mohs 6: Feldspar (steel knife does not easily scratch it)
   - Mohs 7: Quartz (scratches glass)
   - Mohs 8–10: Topaz, corundum, diamond (scratch quartz)

2. **Test on an inconspicuous part** of the specimen. Scratch with each reference material in increasing hardness. Stop when a scratch appears *definitely*.

3. **Record the hardness range.** If the specimen is scratched by Mohs 5 but not Mohs 4, record it as "5."

4. **Cross-check by trying the reverse.** Use the specimen to scratch the reference materials. Consistency strengthens confidence.

### Step 3 — Assess cleavage and fracture

1. **Look for natural breaks or planes** on the specimen. If present, identify the directions:
   - Single plane (one direction)
   - Two planes (two directions, often at a specific angle—e.g., 90° for feldspar, 60°/120° for calcite)
   - Three or more planes (cubic, octahedral, rhombohedral, etc.)

2. **Test cleavage quality** by gently striking the specimen with a hammer on or near a potential cleavage plane. Does it break along a smooth plane or does it fracture irregularly?

3. **Record cleavage geometry** if present:
   - Excellent (clean planes, clear direction)
   - Good (mostly clean, some irregular breaks)
   - Fair (visible planes, many irregular breaks)
   - Poor/absent (no preferred direction, irregular fracture)

4. **Note fracture appearance** where cleavage is absent or poor:
   - Conchoidal (smooth, curved like broken glass)
   - Uneven (irregular, stepped)
   - Splintery (fibrous appearance)
   - Hackly (sharp, ragged edges)

### Step 4 — Determine luster

Luster is how the surface reflects light. It is independent of color.

1. **Tilt the specimen in light** and observe the intensity and character of the reflection.

2. **Classify into one of these categories:**
   - **Metallic** — shiny like polished metal (galena, pyrite, magnetite)
   - **Vitreous** — glassy (quartz, feldspar, tourmaline)
   - **Resinous** — like amber or varnish (sphalerite, some sulfur)
   - **Pearly** — like mother-of-pearl (muscovite, talc on cleavage faces)
   - **Silky** — smooth, like silk fibers (asbestos, satin spar gypsum)
   - **Adamantine** — brilliant like diamond (diamond, corundum, zircon)
   - **Dull/Earthy** — no reflection, matte (chalk, clay minerals, laterite)
   - **Waxy** — like wax or paraffin (jade, some serpentine)

3. **Note if luster varies by face.** Some minerals show different luster on cleavage faces (pearly) versus other surfaces (vitreous).

### Step 5 — Test streak

Streak is the color of the powdered mineral and often differs from the specimen's color.

1. **Use a white unglazed porcelain streak plate** (or the back of a ceramic tile). If unavailable, use unglazed ceramic.

2. **Rub the specimen firmly across the plate,** applying moderate pressure. The powder left behind is the streak.

3. **Record the streak color.** Compare it to the specimen color. Differences are often diagnostic:
   - Red hematite has a red streak; black magnetite has a black streak.
   - Pyrite (brassy) leaves a greenish-black streak.
   - Gold-colored chalcopyrite leaves a black-green streak.

4. **If the specimen is harder than the streak plate** (Mohs 6–10), the streak method is not applicable. Note "no streak" or "harder than plate."

### Step 6 — Synthesize and narrow the candidate list

1. **Create a checklist table** with the five properties:

| Property | Observed Value |
|---|---|
| Color | |
| Hardness | |
| Cleavage/Fracture | |
| Luster | |
| Streak | |

2. **Cross-reference against a mineral table or field guide.** Filter minerals by hardness first (most restrictive), then by cleavage, then by luster, then by streak.

3. **Identify the top 3–5 candidates.** Write down their defining features side-by-side:

| Mineral | Hardness | Cleavage | Luster | Streak | Color |
|---|---|---|---|---|---|
| Candidate A | | | | | |
| Candidate B | | | | | |

4. **Run secondary tests if candidates remain ambiguous:**
   - **Magnetism:** Use a hand magnet. Magnetite is strongly attracted.
   - **Acid reaction:** Drop 10% HCl on the specimen. Carbonates (calcite, dolomite) effervesce.
   - **Specific gravity (rough):** Heft the specimen. Is it notably heavy for its size? (Galena, barite, hematite are denser than quartz.)
   - **Double refraction (birefringence):** Look through transparent specimens at a line beneath. Do you see two lines? (Calcite shows this strongly.)

5. **Document the final identification** with confidence reasoning (see Evaluation Criteria).

## Evaluation Criteria

| Check | Pass |
|---|---|
| Hardness was measured and recorded as a single Mohs value (or range) | Y/N |
| Cleavage/fracture was observed and described geometrically (planes, angles, quality) | Y/N |
| Luster was classified into one of the eight standard terms | Y/N |
| Streak was performed on appropriate specimens or explicitly noted as "not applicable" | Y/N |
| At least 3 candidate minerals were identified from reference material | Y/N |
| The final identification is supported by ≥3 matching properties (not color alone) | Y/N |
| Secondary tests were run for ambiguous candidates | Y/N |

## Red Flags

- **Color used as the primary identifier.** Color alone is unreliable; hundreds of minerals span the same hue.
- **Hardness estimated visually or by fingernail scratch alone.** Systematic comparison to reference materials is required.
- **Cleavage not checked.** Many minerals are indistinguishable without knowing whether they break into planes or fragments.
- **Luster classified vaguely** ("shiny," "dull"). The eight standard terms are precise and diagnostic.
- **Streak not performed despite specimen being soft enough.** Streak often contradicts visual color and is a key discriminator.
- **Single candidate chosen without checking alternatives.** The probability of the first guess being correct drops sharply if ≥2 other minerals share ≥2 of the observed properties.
- **Specimen is altered, oxidized, or heavily weathered.** Tarnished surfaces show false luster and streak; internal properties become unreliable. Note the specimen's condition.
