# Anchoring

## Overview
Anchoring is the process of linking an internal state (emotion, physiological response, confidence, focus) to an external or internal stimulus (gesture, word, image, sound, physical location, taste). Once established, presenting the stimulus reliably triggers the state. Practitioners use this to diagnose unintended state-triggers (why does that phrase always deflate confidence?), to repair broken anchors (why doesn't this coping technique work anymore?), and to intentionally install new anchors for high-performance or therapeutic contexts. The discipline is precision mapping — most people notice correlations between cues and states but never isolate the *actual* trigger or test whether it holds under variation.

## Analytical Procedure

### Phase 1 — State and Context Identification

1. **Define the target state in sensory terms.** Not "confident" but "chest open, jaw unclenched, voice steady, eyes forward." Ask the subject to recall a concrete moment when they had the state and describe their body, tone, and visual focus. Write these down verbatim.

2. **Identify the current stimulus (if repairing an anchor) or desired stimulus (if installing one).** For repair: "What happens right before this state collapses?" For installation: "What signal do you want to use? A hand gesture? A word? A location?" Be specific. "Touch thumb to forefinger" is a stimulus. "Feel motivated" is not.

3. **Test the link in isolation.** Ask the subject to present the stimulus without the state present. Does the state spontaneously arise? If yes, the anchor is already active. If no, it either needs installation or the stated stimulus is not the real one.

### Phase 2 — Anchor Mapping

4. **Trace the history of the association.** When did this stimulus and state first appear together? Was it:
   - Coincidental (happened once, repeated without design)?
   - Contextual (the stimulus was present many times during a period when the state was active)?
   - Intentional (someone explicitly taught the association)?
   - Traumatic (linked by a single high-intensity event)?
   Mark the origin type. Origins matter because they determine whether the anchor is fragile or resilient.

5. **Map competing anchors.** Are there other stimuli that trigger the same state? Are there other states triggered by the same stimulus? These overlaps matter:
   - If Stimulus A triggers both State 1 (desired) and State 2 (undesired), the anchor is contaminated.
   - If multiple stimuli trigger State 1, the state is over-anchored and may be fragile if one stimulus disappears.
   Create a 2×2 table: Stimulus (A/B/C...) vs. State (1/2/3...). Mark each cell as strong, weak, or absent.

6. **Check context dependency.** Does the anchor fire in all contexts or only specific ones? Test by asking: "Does this work when you're tired? Stressed? In a group? At 2am?" If the anchor is context-dependent, list the contexts where it *does* work. These are clues to the real mechanism.

### Phase 3 — Anchor Strength and Stability Assessment

7. **Test anchor robustness.** Present the stimulus 3–5 times in succession and observe whether the state arises each time with the same intensity. Rate each occurrence on a 0–10 scale. If ratings vary by >3 points, the anchor is unstable — note the variation pattern (fades? spikes then drops? random?).

8. **Identify interference patterns.** What interrupts the anchor between installation and test? Competing sensations, thoughts, other people's reactions, environmental noise. Even if the anchor fires, does something prevent the state from persisting? Example: gesture triggers confidence, but then inner critic voice says "who are you kidding?" and confidence collapses. The gesture worked; the anchor to doubt was stronger.

9. **Test generalization.** If the anchor is installed in one context (office) or state-chain (after exercise), will it fire in others (home, rest)? Weak generalization suggests the anchor is entangled with environmental or physiological context. Strong generalization means the anchor is portable.

### Phase 4 — Anchor Revision or Installation

10. **For faulty anchors: diagnose the break.** Is the original stimulus still present but no longer triggers the state? Has a competing anchor become dominant? Has the state itself become inaccessible? Has context contamination occurred? This determines whether to repair, replace, or reinforce.

11. **For new anchors: select stimulus and installation path.** If installing, choose a stimulus that is:
    - Unique (not triggered by accident in daily life)
    - Portable (works in any context)
    - Controllable (subject can execute it reliably under stress)
    Then establish the association by:
    - Having the subject fully access the desired state (recall a moment, use breathing/posture/memory priming).
    - At peak intensity of the state, introduce the stimulus for 3–5 seconds.
    - Repeat 5–10 times, waiting for state to fully dissipate between repetitions.
    - Test the anchor immediately (stimulus alone → state check).
    - Retest 24 hours later (decay check) and in a different context (generalization check).

## Evaluation Criteria

| Check | Pass |
|---|---|
| Target state is described in sensory, not emotional, terms | Y/N |
| Current/desired stimulus is specific and unambiguous | Y/N |
| Stimulus-state link was tested in isolation (not assumed) | Y/N |
| Origin of the association is identified and tagged (coincidental/contextual/intentional/traumatic) | Y/N |
| Competing anchors are mapped in a 2×2 table | Y/N |
| Anchor robustness tested 3+ times with intensity ratings | Y/N |
| Context-dependency or interference patterns documented | Y/N |
| Installation or repair plan is concrete and sequenced | Y/N |

## Red Flags

- "The anchor mostly works" or "works sometimes." Anchors either fire or they don't. Variability is a failure diagnosis, not a success condition.
- The stimulus and state are not isolated. Asking "Does saying 'you've got this' when standing with hands on hips trigger confidence?" conflates stimulus (the phrase + the posture) — test them separately.
- No test of the anchor in a different context. An anchor that only works in the therapy office may not survive the commute to work.
- The "state" is described as a thought ("remember you're capable") instead of a physiology (jaw relaxed, breath steady, shoulders back). Anchors are neurological, not cognitive.
- Competing anchors are present but unmapped. If a gesture triggers both focus and anxiety, claiming the anchor "works" is premature.
- Installation skipped the dissipation step. Repeating the stimulus and state 10 times in a row creates fatigue, not a reliable anchor. State must fully clear between installations.
