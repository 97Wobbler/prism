# 4-Factor Fair Use Test

## Overview
Fair use is an affirmative defense to copyright infringement that permits limited use of copyrighted material without permission under four statutory factors. The framework does not yield a binary yes/no answer; instead, it requires weighing all four factors together, with emphasis shifting depending on context. Practitioners apply this test when they need to defend a use of copyrighted material, assess litigation risk, or advise clients on the defensibility of a proposed project before launch.

## Analytical Procedure

### Factor 1 — Purpose and Character of the Use

1. **Identify the purpose of the use.** Describe in one sentence what the use accomplishes. Examples: "The work excerpts the original to criticize its political message," "The use transforms the original into educational material for classroom distribution," "The work incorporates the original as decoration without alteration."

2. **Classify the purpose on this spectrum:**
   - **Transformative** — The use adds new expression, meaning, or message not present in the original. The original is a means to the new work's purpose, not the purpose itself.
   - **Derivative but non-transformative** — The use adapts or repurposes the original but the new work's primary value comes from the original material itself.
   - **Republication/Substitution** — The use reproduces or distributes the original largely as-is, competing with the original's market.

3. **Check for commercial purpose.** Is the use sold, licensed, or used to generate revenue? (Commercial use weighs against fair use, but is not dispositive; transformative commercial uses can be fair use.)

4. **Check for education, criticism, commentary, or reporting.** Does the use fall into one of these favored categories? (These weigh toward fair use, but not conclusively.)

**Record:**
- Purpose (one sentence)
- Classification (Transformative / Derivative / Republication)
- Commercial? (Yes/No)
- Favored category present? (Yes/No — which ones)

### Factor 2 — Nature of the Copyrighted Work

1. **Identify what type of work is being used.** Is it a factual work (news, history, data), creative work (novel, song, film), or mixture?

2. **Apply this scale:**
   - **Highly creative** — Fiction, music, film, art. Stronger copyright protection; fair use is harder to establish.
   - **Factual or educational** — News, history, scholarly work, reference material. Weaker copyright protection; fair use is easier to establish.
   - **Functional with creative elements** — Software with creative UI, reference work with original design. Mixed protection.

3. **Note whether the original was published or unpublished.** Unpublished works receive stronger copyright protection; using an unpublished manuscript (even for criticism) weighs against fair use.

**Record:**
- Work type (Highly Creative / Factual / Functional-Mixed)
- Published or unpublished

### Factor 3 — Amount and Substantiality of the Portion Used

1. **Measure quantity:** What percentage or word/byte count of the original was reproduced? (E.g., 200 words from a 80,000-word book = 0.25%; 15 seconds of a 3-minute song = 8%.)

2. **Measure substantiality:** Did the use take the "heart" of the original — the most recognizable, valuable, or defining part? This is qualitative, not merely quantitative.
   - Example: Copying the chorus of a song (substantive) vs. copying a bridge section no listener remembers (less substantive).
   - Example: Copying a famous opening line of a novel (substantive) vs. copying a transition paragraph (less substantive).

3. **Assess necessity:** Was the amount used necessary to achieve the stated purpose from Factor 1? If the purpose was criticism of the original's opening, did you need to quote the entire work?
   - Necessary use → weighs toward fair use.
   - Excessive use → weighs against fair use.

**Record:**
- Quantity (percentage or count)
- Substantiality (Heart of work / Important section / Supporting detail)
- Necessity (Necessary / Partially necessary / Excessive)

### Factor 4 — Effect on the Market for the Original

1. **Identify the market for the original work.** Who buys or licenses it? What is the revenue model? (E.g., book sales, streaming, licensing fees, advertising.)

2. **Assess direct market harm:** Does the use substitute for sales/licensing of the original? Would the user have bought or licensed the original if the fair use version were not available?
   - Strong substitution → weighs heavily against fair use.
   - No substitution → weighs toward fair use.

3. **Assess derivative market harm:** Is the copyright holder prevented from licensing derivative versions (adaptations, translations, educational editions, merchandise) because the fair use use already fills that market?
   - Example: A book excerpt prevents the publisher from selling a "Greatest Hits" collection.
   - Example: A clip prevents licensing the film for educational use.

4. **Assess market impact from the use's nature (not just its existence).** Does the use harm the market by commenting negatively on the original, even without direct substitution?
   - Example: A critical review that discourages purchasing.
   - Example: An exposé that reveals factual errors, reducing demand.
   (This is weak evidence against fair use; commentary is often protected.)

5. **Check for permission/licensing availability:** Could the user have licensed the material? If licenses are not available or are prohibitively expensive, fair use is strengthened. If licenses are readily available at reasonable rates and the user ignored them, fair use is weakened.

**Record:**
- Market for the original (describe briefly)
- Direct substitution? (Yes / No / Partial)
- Derivative market harm? (Yes / No)
- Licensing available? (Yes, reasonable / Yes, prohibitive / No)

## Evaluation Criteria

| Check | Pass |
|---|---|
| Factor 1 purpose is described in one sentence and classified (T/D/R) | Y/N |
| Factor 1 commercial status and favored category identified | Y/N |
| Factor 2 work type assigned (Creative/Factual/Mixed) | Y/N |
| Factor 2 published status noted | Y/N |
| Factor 3 quantity measured (percentage or count) | Y/N |
| Factor 3 substantiality assessed (Heart/Important/Detail) | Y/N |
| Factor 3 necessity evaluated against stated purpose | Y/N |
| Factor 4 original market described | Y/N |
| Factor 4 substitution assessed (direct and derivative) | Y/N |
| Factor 4 licensing status checked | Y/N |
| All four factors weighted and traded off in conclusion | Y/N |

## Red Flags

- Factor 1 purpose is vague or circular ("we used it because we needed it"). Refine it to a concrete end use.
- Factor 3 shows high quantity (>30%) and high substantiality but justification is "we needed all of it." If the original could be summarized or replaced with a smaller excerpt, necessity claim is weak.
- Factor 4 market analysis is absent or treats the market as nonexistent ("nobody buys this"). All copyrighted works have some market; if you cannot identify it, research harder.
- Factors are analyzed in isolation with no trade-off. Fair use requires weighing all four together. A transformative use (Factor 1) can outweigh harm to the market (Factor 4), but the harm still must be acknowledged.
- The analysis assumes all factors point the same direction (all toward fair use or all against). Real cases have mixed signals; if your analysis shows no tension, re-examine.
