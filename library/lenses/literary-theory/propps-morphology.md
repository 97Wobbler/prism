# Propp's Morphology

## Overview
Propp identified 31 minimal units of narrative action (functions) that recur across Russian folktales regardless of surface details. A function is a narrative act defined not by *who* performs it or *how*, but by its structural role in the story — abduction, prohibition, deception, struggle, victory, all the way through restoration and reward. Practitioners use this lens to parse a narrative into its functional skeleton, to spot which functions are present or absent, and to see how variants of the same tale rearrange or omit the same functions. Unlike plot summary, which is concrete and particular, morphology reveals the abstract blueprint.

## Analytical Procedure

### Phase 1 — Identify the Narrative Sequence

1. **Read or view the narrative artifact in full.** Folktales, fairy tales, legends, myths, or modern fiction derived from folklore all work. Summarize the plot in 2-3 sentences for reference.

2. **Locate key turning points and reversals.** Mark moments where the protagonist's circumstances change: separation, loss, gain, conflict, resolution. These are typically where functions occur.

3. **Write out the sequence of major events.** Use plain language. Example: "The youngest daughter is forbidden to enter the tower. She enters anyway. A dragon captures her. The prince battles the dragon. The prince wins and marries the daughter."

### Phase 2 — Map Events to Functions

4. **For each event, assign it to one or more of the 31 Propp functions.** Use the reference table below. Functions are ordered as they typically appear in folktales, but not all tales contain all functions. A single event may map to one function only.

| # | Function | Definition |
|---|---|---|
| 1 | **Initial situation** | Setting and introduction of characters; family life is established |
| 2 | **Absentation** | A character leaves (death, departure, disappearance) creating lack or need |
| 3 | **Interdiction** | A prohibition or command is laid on the protagonist |
| 4 | **Violation** | The protagonist breaks the prohibition |
| 5 | **Reconnaissance** | An antagonist seeks information (spying, eavesdropping, testing) |
| 6 | **Delivery** | The antagonist receives information or something of value |
| 7 | **Trickery** | The antagonist deceives the protagonist (disguise, false promise) |
| 8 | **Complicity** | The protagonist is deceived and agrees to something or opens access |
| 9 | **Villainy / Lack** | The antagonist causes harm (theft, abduction, murder, deprivation) OR a character realizes something is missing |
| 10 | **Mediation / Connection** | The protagonist or victim learns of the villainy; the call to action is delivered |
| 11 | **Counteraction** | The protagonist accepts the call and decides to act (agrees to search, to fight, etc.) |
| 12 | **Departure** | The protagonist leaves home to undertake the quest |
| 13 | **First function of the donor** | The protagonist meets a test, question, or request from a helper figure |
| 14 | **Hero's reaction** | The protagonist passes or fails the test; shows respect or rudeness to the donor |
| 15 | **Acquisition of magical agent** | The donor gives a magical object, advice, or power to the protagonist |
| 16 | **Spatial transference** | The protagonist is transported to another place (flight, journey) |
| 17 | **Struggle** | The protagonist battles the antagonist in direct confrontation |
| 18 | **Marking** | The protagonist receives a scar, mark, or sign of victory |
| 19 | **Victory** | The antagonist is defeated, killed, or overcome |
| 20 | **Liquidation of lack** | The object sought is obtained, the captive freed, the problem solved |
| 21 | **Return** | The protagonist travels back home or to a safe place |
| 22 | **Pursuit** | The antagonist or agent of antagonist chases the protagonist during return |
| 23 | **Rescue** | The protagonist is saved from pursuit by a helper or obstacle that stops the pursuer |
| 24 | **Unrecognized arrival** | The protagonist arrives home but is not recognized or believed |
| 25 | **Unfounded claims** | A false hero (impostor) claims credit for the protagonist's deeds |
| 26 | **Difficult task** | The protagonist is given a difficult task to prove identity or worth |
| 27 | **Accomplishment** | The protagonist successfully completes the difficult task |
| 28 | **Recognition** | The protagonist is recognized and identified (scar revealed, truth told) |
| 29 | **Exposure** | The false hero or antagonist is exposed and punished |
| 30 | **Transfiguration** | The protagonist is transformed (age restored, curse lifted, rank elevated) |
| 31 | **Punishment** | The antagonist receives punishment; often a second protagonist marries the heroine |

5. **For each event in your sequence, write the function number and name next to it.** Example: "The dragon captures her" = Function 9 (Villainy). Some events may be skipped if the tale omits them.

6. **Note any functions that are absent but structurally expected.** Absence is meaningful. A tale with no magical agent (Function 15) is different from one that does.

### Phase 3 — Analyze the Morphological Pattern

7. **Extract the functional skeleton — the sequence of function numbers.** Example: 1, 2, 3, 4, 9, 10, 11, 12, 13, 14, 15, 16, 17, 19, 20, 21, 28, 30, 31.

8. **Check for structural integrity.** Propp noted that certain functions form obligatory pairs or sequences:
   - Functions 3–4 (prohibition + violation) almost always occur together.
   - Functions 9–10 (villainy + mediation) are typically the inciting incident.
   - Functions 13–15 (donor test + reaction + magical agent) form a unit.
   - Functions 17–19 (struggle + marking + victory) form a battle unit.
   
   Mark which units are present, broken, or rearranged.

9. **Compare to known tale variants.** If analyzing a folktale, look for published variants (e.g., different versions of "Cinderella" or "Snow White"). Are the function sequences identical? Where do they diverge? Divergences reveal what is essential (the functions) versus what is decoration (character names, setting details).

## Evaluation Criteria

| Check | Pass |
|---|---|
| All major story events have been mapped to functions | Y/N |
| The functional sequence is ordered chronologically | Y/N |
| Obligatory function pairs (e.g., 3–4, 9–10) are correctly identified | Y/N |
| Absent but structurally expected functions are noted | Y/N |
| At least two structural units (donor test, battle, recognition) are identified | Y/N |
| If comparing variants, key divergences in function order are noted | Y/N |

## Red Flags

- Every event maps to a different function with no clustering. Propp's functions naturally cluster into units; if yours don't, you may be over-parsing or misidentifying function boundaries.
- Functions 13–15 (donor sequence) are marked as absent without explanation. This sequence is near-universal in folktales; its absence suggests either a non-folktale genre or a missed function.
- The functional sequence is heavily non-linear (e.g., 1, 2, 31, 3, 9, 20). While some rearrangement happens, wildly out-of-order sequences often indicate misidentification of functions or a narrative that doesn't follow folktale logic.
- Unrecognized arrival (Function 24) and false hero (Function 25) are marked present but no actual deception occurs in the plot. These are distinct and easy to confuse with simple non-recognition.
- The analysis stops after Phase 1 (event listing) without mapping to functions. The point of Propp is the morphology, not plot summary.
