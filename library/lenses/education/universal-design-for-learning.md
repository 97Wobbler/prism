# Universal Design for Learning (UDL)

## Overview
Universal Design for Learning audits the three primary means by which learners engage with material — how they receive information (representation), how they act on what they've learned (action and expression), and what drives their effort (affect and engagement) — and identifies points where the design unnecessarily blocks certain learners. The lens is structural, not remedial: it asks whether the course design assumes a single way to learn and what happens to learners who can't or won't use that path. Practitioners use this when designing new courses, retrofitting materials, or debugging why particular students chronically disengage.

## Analytical Procedure

### Phase 1 — Map the Three Networks

1. **Identify the target learning outcome.** Write it in one sentence: what should a learner be able to do, understand, or create by the end? Example: "students will analyze primary sources to construct an historical argument."

2. **For Representation (How learners receive information):**
   - List every modality in which the core content is presented: text, video, audio, image, lecture, lab demonstration, simulation, etc.
   - For each modality, note whether it is **required** (the only path to the information) or **optional** (alternative exists).
   - Identify which modalities assume specific sensory or cognitive abilities: visual language, hearing, reading fluency, prior vocabulary, working memory load, etc.
   - Mark each assumption as **hard** (cannot be removed) or **soft** (removable with design).

3. **For Action & Expression (How learners demonstrate learning):**
   - List every form of assessment or demonstration: written exam, oral presentation, portfolio, lab report, group project, multiple choice, essay, performance, etc.
   - For each form, note what physical/cognitive/social capability it demands: writing speed, speech clarity, standing/sitting duration, collaboration tolerance, public speaking, fine motor control, etc.
   - Mark which assessments are **aligned to the outcome** (what they measure is actually what you wanted to measure) and which measure **compliance with the format** (students who understand the concept may fail because they can't produce it in the required form).

4. **For Affect & Engagement (What sustains effort):**
   - List moments in the course where learners make autonomous choices: topic selection, pace, grouping, media type, deadline, presentation style, etc.
   - Identify which choices are **genuine** (real alternatives with equal validity) and which are **staged** (all paths lead to the same constraint — e.g., "choose your own topic, but it must be a 5-page essay").
   - Note what the course communicates about who belongs: Are diverse learners visibly represented in examples? Are identities affirmed or erased in case studies and language? Are errors treated as learning moments or as deficits?

### Phase 2 — Identify Barriers

5. **For each modality and assessment form, ask: "Who is blocked?"** Do not ask "Is this accessible?" (too vague). Ask specifically:
   - Deaf learners: Is there caption/transcript/sign interpretation of audio?
   - Blind learners: Is visual content described in text? Can they navigate interactive media?
   - Dyslexic learners: Is there text-to-speech or reduced reading load? Are fonts sans-serif?
   - Non-native speakers: Is jargon defined? Are idioms avoided?
   - Learners with low motor control: Can they opt out of handwriting, use speech-to-text?
   - Learners with ADHD: Are there checkpoints, summaries, and external structure?
   - Learners from underrepresented backgrounds: Do examples and role models include people who look like them?
   - Learners who learn by doing: Does the course offer labs, simulations, projects, or only passive reception?

6. **Distinguish barriers from legitimate difficulty.** A barrier is something that blocks a learner who otherwise has the ability. A legitimate challenge is core to the outcome (e.g., "this is a writing class, so fluent writing is the outcome; we can't eliminate that demand"). Mark each barrier as **removable** (design can be changed), **partially removable** (core to outcome but can be scaffolded), or **non-removable** (essential to the learning goal).

### Phase 3 — Generate Alternatives

7. **For each removable barrier, propose an alternative modality or assessment form that achieves the same outcome.** Do not propose "easier" versions — propose *different* versions:
   - Instead of: "written essay"
   - Consider: "recorded explanation," "visual argument (poster, infographic)," "group presentation," "annotated podcast," "interactive walkthrough"
   - Check: Does this alternative require the same underlying understanding? Yes/no for each option.

8. **For partially removable barriers, propose scaffolds that maintain rigor while lowering the barrier to entry:**
   - Graphic organizers before writing
   - Model answers before open-ended work
   - Dictation software alongside written assignments
   - Flexible pacing within a semester
   - Closed captions on all videos (not optional)

9. **For engagement, audit optionality.** Rewrite each "choice" to be genuine. Example:
   - Bad: "Choose a topic for your project (must be 2000–2500 words, submitted by Friday, graded on the rubric)"
   - Better: "Choose a topic and a format: 2000–word essay, 15-min video, annotated podcast, interactive website. Submit by Friday. Graded on the same rubric applied to each format."

## Evaluation Criteria

| Check | Pass |
|---|---|
| Target learning outcome stated in one sentence | Y/N |
| All modalities for content delivery listed with required/optional marking | Y/N |
| Each modality interrogated for sensory/cognitive assumptions | Y/N |
| All assessments listed with compliance-vs.-outcome distinction made | Y/N |
| Engagement choices identified as genuine or staged | Y/N |
| Barriers identified specifically (not "make it accessible") for at least 3 learner groups | Y/N |
| Each barrier classified as removable/partially removable/non-removable | Y/N |
| At least one alternative modality proposed for a removed barrier | Y/N |

## Red Flags

- All barriers are marked "non-removable." This usually means the analysis didn't distinguish barriers from the core outcome. A reading class requires reading; it does not require *written* exams or visual text display.
- Representation alternatives all reduce cognitive load. UDL is not about making things easier; it's about offering equivalent rigor in different forms. If all alternatives are "simplified," the design has failed.
- Engagement choices exist but all paths converge on the same constraint. ("You can work alone or in groups, but everyone submits a 5-page essay by Friday.") The choice is staged.
- No barriers identified for learners from underrepresented backgrounds. Barriers are not only sensory/motor. Language, examples, and identity affect belong in the audit.
- Scaffolds are presented as "for struggling students" rather than "for all learners." Scaffolds should be universal; framing them as remedial stigmatizes them.
